# YouTube 한글 자막 변환 (YouTube Korean Subtitle)

클릭 한 번으로 유튜브 동영상의 자막을 한국어(자동 번역)로 변환해주는 크롬 확장 프로그램입니다. 유튜브의 기본 자동 번역 기능을 활용하여, 여러 번 클릭해야 하는 번거로운 과정을 단일 버튼 클릭으로 자동화합니다.

## ✨ 주요 기능

- **원클릭 한글 자막**: 유튜브 플레이어 하단 컨트롤 바에 생성되는 `[가]` 버튼을 클릭하면 즉시 한국어 자막이 활성화됩니다.
- **자동화된 프로세스**: `설정 > 자막 > 영어(자동 생성됨) > 자막 > 자동 번역 > 한국어`로 이어지는 복잡한 단계를 자동으로 처리합니다.
- **토스트 알림**: 자막 활성화 및 비활성화 상태, 혹은 자막 미지원 영상에 대한 안내를 우측 하단 토스트 메시지로 직관적으로 알려줍니다.
- **SPA 완벽 지원**: 유튜브의 페이지 이동(SPA 네비게이션) 방식에서도 문제없이 버튼이 삽입되고 동작합니다.

## 📦 설치 방법 (개발자 모드)

이 확장 프로그램은 크롬 웹 스토어에서 공식 다운로드하지 않고, 로컬에서 직접 설치하여 사용할 수 있습니다.

1. 이 저장소를 클론하거나 ZIP 파일로 다운로드하여 압축을 풉니다.
2. 구글 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력하여 접속합니다.
3. 우측 상단의 **개발자 모드(Developer mode)** 토글을 켭니다.
4. 좌측 상단의 **압축해제된 확장 프로그램을 로드합니다(Load unpacked)** 버튼을 클릭합니다.
5. 다운로드 받은(또는 클론한) `youtube-korean-subtitle` 폴더를 선택합니다.
6. 설치가 완료되면 확장 프로그램 목록에 표시됩니다.

## 🚀 사용법

1. 아무 유튜브 동영상이나 재생합니다.
2. 플레이어 하단 우측(설정 아이콘 옆)에 새로 생성된 **[가]** 아이콘 버튼을 확인합니다.
3. 버튼을 클릭하면 잠시 후 한국어 자막이 자동으로 활성화됩니다. 
4. 다시 버튼을 클릭하면 자막이 꺼집니다(토글 기능).

> **참고**: 원본 동영상에서 자체적으로 어떠한 자막도 제공하지 않는 경우(자동 생성 자막 포함), 이 확장 프로그램은 작동하지 않습니다.
